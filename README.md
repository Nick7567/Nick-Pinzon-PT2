# Nick-Pinzon-PT2
##Como configurar IsarVDI
Primero tenemos que ir a IsarVDI y crear un escritorio nuevo.

Ponemos el nombre que queramos y seleccionamos la maquina virtual que queramos utilizar.

Bajamos y le damos a "crear"

Despues, le damos a los tres puntos de la nueva maquina virtual y le damos al "lapiz" 

Aqui editaremos nuestra maquina

En "vCPUS" ponemos 4

En "Memoria (GB)" ponemos 16

En "Hard Disk" ponemos Default

En "Redes" ponemos "puigcastellar1"

##Comprobar que el NextCloud funciona correctamente
Entramos a nextcloud y ahi iniciamos sesion con normalidad

Despues, nos vamos a "files"
